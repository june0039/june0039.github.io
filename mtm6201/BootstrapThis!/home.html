About Bootstrap

Open Source

Bootstrap is an open source tool kit for the developing with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with our SASS variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery. It works great for your homework too! You can use codes in your html to create beautiful, responsive code and no need for a style sheet! That's right it can all be done in html with the power of SASS and Bootstrap. Bootstrap has done most of the work for you with precoded features and components. Keep on reading to find out about some of those built in features and they work. There is also extensive instructions as to how to use anything on Bootstrap.com right with all the other features.

Mobile First

Bootstrap includes a powerful mobile flexbox grid to build layouts of all shapes and sizes thanks to a twelve column system, five default responsive tiers, Sass variables and mixins, and dozens of predefined classes. You can use margins and padding with a responsive grid, plus alignment. You can use from size extra small-xsm to extra large-xl to define the screen size to display your attributes Here is one example: text-lg-center programs the page to center the text at a large size screen. If I wrote text-sm-cen the page would display text in the center of the page from the small screen up unless otherwise specified. You can say offset-lg-0 which will then take away the offset at the begining of the large screen, min-width large screen. The Bootstrap sizes start as follows: xsm-:less than 576px sm-:@ 576px md-: less than to @ 768px lg-less than to @ 992px xl-less than to @ 1200px.

Components Library

Bootstrap comes with a very extensive library that includes buttons and lists to carousels and modals. Components are easy to integrate into any site and require no knowledge of JavaScript to use. Made easy for everyday developer! Creativity built in! You just have to look up the component you want to use on Bootstrap.com under components. Then once you find the look of what you want you just hit copy the button to copy the code for that component. Add it to your program and it is complete! You can still make changes to things like margins, padding, location, position on your page with their coding.

Alerts

Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages. This is very handy and can be easily displaye of colors using the codes given on Bootstrap.com. Alerts can hold any size of text. They also have an optional dismiss button and alerts come in all the same colors as text.

Breadcrumbs

Indicate the current page's location within a navigational hierarchy that automatically adds separators via CSS. Goodnews tho, the seperators can be changed. Breadcrumbs provide navigation so it is best practice to label them with meaningful names. Ones that describes where the navigation is going.

List Group

List groups are a flexible and powerful component for displaying a series of content. A basic list group is a ul- unordered list group. Like in html coding, it displays just a list with no numbers. You can add active as a feature to a unordered list and it will highlight the current active selection. You can also add disabled as a feature and it will display the item as non active and not available to click.
